- 常用的三角函数的关系

$$
\sin x = \frac{b}{c}, &\cos x = \frac {a}{c}\\
\tan x = \frac{b}{a},& \cot x = \frac {a}{b} \\
\tan x = \frac {\sin x}{\cos x}, &\cot x = \frac{\cos x}{\sin x}   \\
\tan x = \frac{1}{\cot x} ,&  \cot x = \frac{1}{\tan x} \\
\sec x = \frac {1}{\cos x }, & \csc x  =\frac {1}{\sin x}
$$
